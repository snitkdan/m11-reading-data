Welcome to Module 11 Exercise 1

To complete this exercise, edit your main.js file. Your aim is to understand how the $.get function reads data into a web application. You'll also get practice building a table with jQuery.

  • Read in your data/students.csv using the $.get function
  • Use Papa.parse to parse your data into a usable format
  • Create an HTML table element using jQuery
  • Append a table header (<th>) for each key in your first observation
  • Iterate through your array and create a new table row (<tr>) for each element in your array
  • Add a cell (<td>) for each key/value pair in your object
  • Select your section with id sandbox section, and append your table to it
